## 功能特性

- **虚拟组网** - 轻松创建安全的虚拟局域网
- **设备管理** - 查看在线/离线设备，支持设备分组
- **内网访问** - 内置浏览器访问本地内网服务（用户端/管理端双入口）
- **流量统计** - 实时显示网络速度和流量使用情况
- **主题切换** - 支持浅色/深色/跟随系统三种模式
- **系统托盘** - 最小化到托盘，快速连接/断开

## 内网访问功能

应用内置 WebView 浏览器，支持访问本地内网服务：

- **用户端** - 访问用户查询服务（默认: http://127.0.0.1:8080）
- **管理端** - 访问后台管理服务（默认: http://127.0.0.1:8081）
- 支持独立配置服务器地址和端口
- 支持文件上传功能
- 支持页面返回和关闭操作
